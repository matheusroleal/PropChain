Welcome!

This is a file transaction service using ethereum network. By using this service, you will be able to create any relationship between two address and a transaction

In orrder to maintain data imutability, we store the data in a WORM database. Write once read many (WORM) describes a data storage device in which information, once written, cannot be modified. This write protection affords the assurance that the data cannot be tampered with once it is written to the device. On ordinary (non-WORM) data storage devices, the number of times data can be modified is limited only by the lifespan of the device, as modification involves physical changes that may cause wear to the device. The "read many" aspect is unremarkable, as modern storage devices permit unlimited reading of data once written.

In this website you can create a transaction, list and show the data store in the transaction.
